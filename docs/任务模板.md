- [简要说明背景与目标]

- [范围/约束/风险/依赖]

### 任务要求
- [质量/性能/安全/兼容/扩展/维护/可读/验证/文档等要求]

- [ ] 步骤1

- [ ] 步骤3

### 验证与交付
- [测试类型/环境/基线]

- YYYY-MM-DD: [关键进展/决策/问题]
